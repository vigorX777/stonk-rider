# Changelog

## 1.3.2 - 2026-06-13

- 优化跳跃手感：空格跳跃改为任一车轮着地即可触发，不再要求后轮着地。
- 新增 140ms 跳跃输入缓冲，落地前轻微提前按空格会在接触地面的第一帧起跳。
- 复位时清空未消费跳跃输入，避免摔车刷新后被旧输入带起跳。

## 1.3.1 - 2026-06-13

- 项目品牌迁移为中文名「韭菜骑士」，英文名使用 `Leek Knight`。
- 对外链接、README、页面标题、首页品牌、localStorage key、包名、Wrangler Pages 项目名统一切换到 `leek-knight`。
- 试玩域名更新为 `https://leek-knight.pages.dev`。

## 1.3.0 - 2026-06-13

- 新增 Cloudflare Pages Functions 排行榜 API，支持提交成绩、查询单股榜和全局榜。
- 新增 D1 `leaderboard` schema、Wrangler Pages 配置和本地 SQLite/D1 契约测试。
- 同一匿名玩家同一股票只保留最高收益率；全球榜按玩家跨股票最佳成绩去重。
- 后端提交校验绑定固定 20 支 A 股快照代码与名称，拒绝非完整通关、异常收益率、非法昵称和金额不一致数据。
- 通关弹窗加入昵称输入、随机昵称、提交成绩、排名结果、Top 3 和完整排行榜。
- 完整排行榜支持股票筛选，并高亮当前浏览器玩家的成绩。
- README 增加 Cloudflare Pages + D1 首次部署、本地 Pages Functions 验证和更新部署说明。
- 本地 Pages Dev 验证使用 `wrangler.toml` 的 D1 绑定，避免 `--d1=DB` 临时库与 schema 初始化目标不一致。

## 1.2.0 - 2026-06-13

- 新增双 Combo 系统：涨跌独立计数，吃涨断跌、吃跌断涨，各自 6 秒窗口，乘数最高封顶 `x2.5`。
- 新增真实跳跃：空格键触发，最初为后轮着地条件，跳跃力随速度递增并封顶约 `0.10`；1.3.2 已调整为任一车轮着地即可跳跃。
- 未吃金币持续保留在赛道上，玩家可以倒车回头拾取，不再受结算索引窗口限制。
- 后轮落地后牵引力在 200ms 内从 15% 渐入到 100%，hit-stop 期间暂停渐入计时。
- 结算瞬间新增 hit-stop、动态屏幕震动、全屏闪色和定向粒子，让金币反馈更强。
- 新增双 Combo HUD，涨跌两槽同排显示，按 `NICE`、`GREAT`、`INSANE` 分级强化活跃槽。
- 余额数字加入老虎机式滚动和弹性过冲，结算 Toast 按金额冲击力分级展示。
- 金币音效改为 combo 感知序列，高连击时增加更密集音阶与高频泛音；切换、超时或摔车断连时播放独立下坠音效。
- 补充 Combo 纯函数、双槽互斥、结算组合和跳跃力测试。
- 修复运行页尚未显示时创建 Phaser 场景导致画布尺寸为 `0×0` 的初始化时序问题。

## 1.1.4 - 2026-06-13

- 金币从“越过 x 坐标自动结算”改为严格拾取触发，必须车轮、车身或头盔靠近金币才会结算。
- 默认金币间隔从 5 个交易日调整为 8 个交易日，分布更稀疏，并新增需要冲坡/腾空获取的高位金币。
- 涨幅金币显示 `+`，跌幅金币显示 `-`，继续遵循 A 股红涨绿跌颜色习惯。
- 吃金币音效改为更强的多音阶金币序列，收益段更尖锐明亮，亏损段更低沉下坠。
- 错过金币不会阻塞通关；吃到金币时，金额变化仍只使用该金币对应的原始行情区间涨跌幅。

## 1.1.3 - 2026-06-13

- 修复左右键姿态控制被刚性轮轴约束抵消的问题，长按 A/左键持续翘头，长按 D/右键持续下压。
- 加入姿态角速度目标与更明显的前后轮载荷转移，并在 HUD 实时显示当前姿态输入。
- 修复 Web Audio 解锁竞态，进入车库和开始骑行时会等待音频上下文真正启动。
- 提高引擎与事件音效的基础响度和频率表现，并暴露音频运行状态供浏览器回归验证。
- 音频恢复增加超时与能力检测，音频不可用或被浏览器挂起时不再阻塞选股和开局。
